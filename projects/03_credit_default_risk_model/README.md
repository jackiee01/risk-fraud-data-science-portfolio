# Credit Default Risk Model

## Overview

This project will focus on predicting whether a customer is likely to default on a loan or credit obligation.

## Business Problem

Credit risk models help financial institutions assess borrower risk and support lending decisions.

The goal is to build a classification model that predicts default risk and translates model outputs into risk-based customer segments.

## Domain

Credit Risk / Finance

## Problem Type

Classification

## Planned Workflow

1. Load and inspect credit risk data
2. Analyze default distribution
3. Identify important borrower and loan-related features
4. Handle missing values and categorical variables
5. Train classification models
6. Evaluate model performance
7. Create risk bands
8. Interpret results for credit risk decision-making

## Planned Models

- Logistic Regression
- Decision Tree
- Random Forest

## Key Metrics

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
- PR-AUC
- Confusion Matrix
