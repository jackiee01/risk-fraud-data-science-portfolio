# Machine Learning Practice

This folder contains machine learning practice focused on supervised learning, model evaluation, and risk-related prediction tasks.

## Focus Areas

- Train/test split
- Logistic Regression
- Decision Tree
- Random Forest
- Classification metrics
- Confusion Matrix
- Precision, Recall, and F1-score
- ROC-AUC and PR-AUC
- Threshold tuning

## Purpose

The goal is to build core machine learning skills for junior data scientist roles in finance, fraud detection, credit risk, and insurance analytics.
