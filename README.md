## Current Focus

My current focus is building applied data science projects for banking, fraud detection, credit risk, and insurance risk analytics.

The main project currently in progress is Transaction Fraud Detection, which demonstrates a fraud analytics workflow using synthetic transaction data, feature engineering, classification models, and threshold analysis.

## Project Philosophy

The goal of this portfolio is not only to build machine learning models, but also to connect data analysis and model outputs to business decisions.

For finance, insurance, fraud, and risk analytics, model performance must be interpreted together with business trade-offs such as false positives, missed risk cases, operational review cost, and customer impact.
